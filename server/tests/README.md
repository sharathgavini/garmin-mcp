# Server Tests

This directory contains Node tests for the TypeScript MCP server.

## Coverage

- Date filtering.
- Tool input validation.
- Latest-data tools.
- Activity stream lookup, filtering, and downsampling.
- Workout analysis helpers.
- Archive partition range queries.
- Sync lock behavior.
- OAuth registration, authorization, token issuing, and protected route behavior.

## Run

```bash
cd server
npm test
```
